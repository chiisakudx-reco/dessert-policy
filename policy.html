基本方針

Dessert（以下「本アプリ」）は、すべてのデータをお使いのデバイス内にのみ保存します。外部サーバーへの送信・クラウドへの保存は一切行いません。通常の使用においてインターネット接続は不要です。

本アプリの開発者 chiisakudx（以下「開発者」）は、ユーザーのプライバシーを最大限に尊重します。本ポリシーは、本アプリがどのようにデータを扱うかを説明するものです。

パスワードは Android の EncryptedSharedPreferences（AES-256）を使用して暗号化保存されます。アプリ画面上で平文表示されることはありません。

PINコードによるアプリロックで、第三者のアクセスからデータを保護します。PINはお使いのデバイスのロック画面とは独立して設定されます。

PINを忘れた場合：アプリ内の「PINを忘れた場合」から、アプリ内のすべてのデータを削除してリセットすることができます。この操作は取り消せません。重要なデータは事前にエクスポート機能でバックアップすることを強く推奨します。

Passwords are encrypted using Android's EncryptedSharedPreferences (AES-256). They are never displayed in plain text within the app.

If you forget your PIN: You can reset the app by deleting all app data through the "Forgot PIN" option. This action cannot be undone. We strongly recommend exporting a backup before this occurs.

データ・プライバシー

データについて

Q. データはどこに保存されていますか？
すべてのデータはお使いの端末内にのみ保存されます。外部サーバーやクラウドへの送信は一切行いません。インターネット接続も不要です。
Q. アプリをアンインストールするとデータは消えますか？
はい、アンインストールするとすべてのデータが削除されます。アンインストール前に設定画面からエクスポート（バックアップ）することをお勧めします。
Q. 機種変更・端末の初期化をした場合、データはどうなりますか？
データは端末内にのみ保存されているため、機種変更・初期化を行うと引き継がれません。事前に設定画面からJSONファイルとしてエクスポートし、新しい端末でインポートすることでデータを移行できます（おやつ代を支援プラン限定）。
セキュリティ

セキュリティについて

Q. PINを忘れた場合はどうなりますか？
PIN認証画面に「PINを忘れた場合」のオプションがあります。ただし、PINを忘れた場合の復元手段はなく、すべてのデータを削除してリセットすることになります。PINは必ずメモしておくか、安全な場所に保管してください。
Q. パスワードはどのように保護されていますか？
パスワードはデバイスのセキュアストレージ（Android: EncryptedSharedPreferences / iOS: Keychain）に暗号化して保存されます。アプリのデータベースにパスワードの実体は保存されません。
Q. 生体認証（指紋・Face ID）には対応していますか？
現在は対応していません。PINコードによる認証のみご利用いただけます。
バックアップ

バックアップについて

Q. 機種変更時にデータを引き継ぐ方法は？
以下の手順で移行できます（おやつ代を支援プラン限定）。
1. 旧端末の設定画面 → エクスポートでJSONファイルを保存
2. ファイルを新端末に転送（メール・クラウドストレージ等）
3. 新端末のDessertで設定画面 → インポートを実行
Q. バックアップファイルにはパスワードも含まれますか？
はい、エクスポートしたJSONファイルにはパスワードを含むすべてのデータが含まれます。ファイルの管理・取り扱いには十分ご注意ください。

機能

機能について

Q. 外部認証アカウントと個別アカウントの違いは何ですか？
Dessertではアカウントを2種類に分けて管理します。
外部認証アカウント：Google・LINE・Appleなど、他のサービスへのログインに使う認証元アカウントです。これに紐づく連携サービスをまとめて管理できます。
個別のサービスアカウント：サービス固有のID・パスワードを持つアカウント（例: Amazonに直接登録したアカウント）です。
この2層構造により「どのGoogleアカウントで何にログインしているか」が一目でわかります。
Q. タグランチャーとは何ですか？
タグを使ってアカウントをグループ化し、ワンタップでURLを開いたりIDをコピーしたりできる機能です。仕事・プライベート・副業など、用途別にまとめておくと素早くアクセスできます。タグランチャーの閲覧・起動は無料でご利用いただけます（タグの作成はおやつ代を支援プランが必要です）。
Data & Privacy

About Your Data

Q. Where is my data stored?
All data is stored exclusively on your device. Nothing is sent to external servers or the cloud. No internet connection is required.
Q. Will my data be deleted if I uninstall the app?
Yes, uninstalling the app deletes all data. We recommend exporting a backup from Settings before uninstalling.

Security

About Security

Q. What happens if I forget my PIN?
You can tap "Forgot PIN" on the lock screen, but there is no recovery option — this will delete all app data and reset the app. Please keep your PIN in a safe place.

Passwords are encrypted and stored in the device's secure storage (Android: EncryptedSharedPreferences / iOS: Keychain). The actual password values are never stored in the app's database.
Q. Is biometric authentication (fingerprint / Face ID) supported?
Not currently. Only PIN code authentication is available at this time.

2. Transfer the file to your new device (via email, cloud storage, etc.)
3. On your new device: Settings → Import
Q. Does the backup file include passwords?
Yes, the exported JSON file includes all data, including passwords. Please handle the file with care.

Features

About Features

Q. What's the difference between External Auth and Individual accounts?
Dessert organizes accounts into two types:
External Auth Accounts: Authentication providers like Google, LINE, or Apple that you use to log into other services. Linked services are grouped under these accounts.
Individual Service Accounts: Accounts with their own unique ID and password (e.g., an Amazon account registered directly).
This two-layer structure makes it easy to see which account you use to log into which services.
Q. What is the Tag Launcher?
The Tag Launcher lets you group accounts by tag and quickly open URLs or copy IDs with a single tap. Organize by work, personal, or side projects for fast access. Viewing and launching is free; creating tags requires the Buy Me a Coffee plan.
